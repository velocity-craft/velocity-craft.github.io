OmniWake Política de Privacidade

Data de vigência: 16 de abril de 2026

OmniWake (doravante "o Aplicativo") valoriza suas informações pessoais e estabeleceu a seguinte política de privacidade em conformidade com as leis de proteção de dados aplicáveis.

1. Informações Pessoais Coletadas

O Aplicativo pode coletar as seguintes informações para fornecer seus serviços.

  • Configurações de alarme: Todos os dados de alarme (horários, etiquetas, sons e configurações) são armazenados apenas no seu dispositivo e não são transmitidos para nenhum servidor externo.
  • E-mail da conta Google: Coletado opcionalmente ao fazer login com Google para sincronizar assinaturas entre dispositivos.
  • ID de publicidade: Usado pelo Google AdMob para exibir anúncios personalizados (apenas na versão gratuita).
  • Gravações de áudio: O áudio capturado ao usar o recurso de gravação de sons de alarme é armazenado apenas localmente no seu dispositivo.
  • Informações do dispositivo: O modelo do dispositivo e a versão do sistema operacional podem ser coletados para relatórios de erros e melhoria do serviço.

2. Finalidade da Coleta e Uso

  • Função de alarme: Os dados de alarme são usados exclusivamente para agendar e acionar alarmes no seu dispositivo.
  • Sincronização de assinatura: Seu e-mail do Google é usado para verificar e sincronizar o status da assinatura Pro entre dispositivos.
  • Publicidade: Anúncios personalizados são exibidos via Google AdMob para usuários da versão gratuita.
  • Melhoria do aplicativo: Dados de erro anonimizados e estatísticas de uso podem ser coletados para melhorar a estabilidade do serviço.

3. Período de Retenção

  • Dados de alarme: Armazenados apenas no dispositivo. Excluídos quando você remove alarmes ou desinstala o Aplicativo.
  • Gravações de áudio: Armazenadas apenas no dispositivo. Excluídas quando você remove o som de alarme associado ou desinstala o Aplicativo.
  • Dados de assinatura: Retidos pelo tempo necessário para verificar o status da assinatura.
  • Dados publicitários: Sujeitos à política de privacidade do Google.

4. Compartilhamento com Terceiros

O Aplicativo não compartilha suas informações pessoais com terceiros em princípio. No entanto, as seguintes exceções se aplicam:

  • Quando você deu consentimento prévio.
  • Quando exigido por lei.
  • Informações limitadas (ID de publicidade, informações do dispositivo) podem ser compartilhadas com o Google AdMob para serviços publicitários.
  • Seu e-mail do Google pode ser compartilhado com nosso serviço de verificação de assinatura (AWS) para confirmar o status Pro.

5. Permissões do Aplicativo

O Aplicativo usa as seguintes permissões:

  • Agendar alarme exato (SCHEDULE_EXACT_ALARM): Para disparar alarmes no horário exato que você definir [Obrigatório]
  • Definir alarme (SET_ALARM): Para criar e gerenciar alarmes no seu dispositivo [Obrigatório]
  • Internet (INTERNET): Para exibir anúncios e verificar assinatura [Obrigatório]
  • Executar na inicialização (RECEIVE_BOOT_COMPLETED): Para restaurar automaticamente seus alarmes após a reinicialização do dispositivo [Obrigatório]
  • Notificações (POST_NOTIFICATIONS): Para notificações de alarme e lembretes [Opcional]
  • Microfone (RECORD_AUDIO): Para gravar sons de alarme personalizados [Opcional]
  • Isenção de otimização de bateria (REQUEST_IGNORE_BATTERY_OPTIMIZATIONS): Para entrega confiável de alarmes em dispositivos com gerenciamento agressivo de bateria [Opcional]

As permissões opcionais são solicitadas apenas quando o recurso correspondente é usado. Você pode usar os recursos básicos do Aplicativo sem concedê-las.

6. Armazenamento e Segurança de Dados

O Aplicativo implementa as seguintes medidas para garantir a segurança de suas informações:

  • Todos os dados de alarme são armazenados no armazenamento interno do dispositivo e não são sincronizados automaticamente com nenhum serviço em nuvem.
  • O backup e restauração dos dados de alarme são iniciados pelo usuário.
  • Conexões criptografadas (HTTPS) são usadas para todas as comunicações de rede.
